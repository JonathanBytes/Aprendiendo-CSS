Selectores simples

Dentro de esta categoría se encierran los siguientes selectores.

Selectores elementales

Selector Universal

Consisten en un *: 


    *{
      background-color: pink;
    }
El estilo se aplicará a todos los elementos. No es recomendable usarlo para cosas que se ven, como por ejemplo un margin: 0.

Selector de tipo o etiqueta

Consisten en usar el nombre de la etiqueta a la cual queremos aplicarle el estilo: 


    h1{
      background-color: yellow;
    }
Es un selector que se usa en los reseteos de estilos o también llamado normalizar estilos

Selectores de atributo

Selector de id

Se usa el id del elemento, este es un atributo dentro de la etiqueta HTML:


      <h1 id="title">Título</h1>

Para seleccionar el id de este elemento h1 se haría lo siguiente en CSS:


    #title{
      background-color: lightcian;
    }

No es recomendable usar el selector de id para aplicar estilos, debido a la especifisidad de este.

Selector de clase

Se utiliza la clase del elemento, es el más recomendable a usar, su uso es similar al uso de las id.


      <h1 class="title">Título</h1>

Para seleccionar la clase de este elemento h1 se haría lo siguiente en CSS:


    .title{
      background-color: lightcian;
    }

Otros atributos

Para estos atributos se utilizan las llaves: []
Al aplicar estilos de esta forma logramos aplicar estilo a todos los elementos que contengan el atributo que hemos indicado dentro de las llaves. Por ejemplo: 


    [href]{
      color: blue;
    }

Sin embargo se puede utilizar de otras formas, como por ejemplo incluir el valor del atributo, de esta forma:


    [href="https//google.com"]{
      color: black;
    }

Es importante resaltar que el valor del atributo debe estar entre comillas.
También se puede incluir una condición, que compruebe si el atributo empieza por ese valor, de esta forma: 


    [href^=valor]{
      color: yellow;
    }

Este fragmento de CSS comprobará si existe un elemento con el atributo href que su valor empiece por valor, por ejemplo, el atributo href=valor-imagen será escogido para aplicar el estilo, sin embargo, los siguientes casos no son válidos:


    <h1 href=test-valor-imagen>Test title</h1> 
    <a href=hola-valor>Link test</a>
