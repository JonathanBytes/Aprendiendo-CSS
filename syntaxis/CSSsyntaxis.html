Sintaxis de CSS

Para aplicar estilos se utiliza la siguiente sintáxis:

Selector

El elemento al que vamos a aplicar estilos


    selector{ /* Regla */
      propiedad:valor; /* declaración */
      propiedad2:valor;
      propiedadN:valor;
    }

Se pueden agregar tantas propiedades como el estilo lo necesite, siempre separadas por ;. Es posible dejar la última propiedad sin punto y coma, pero es mejor acostumbrase a ponérselo a todas

Propiedades

Lo que vamos a cambiar.

Valor

El nuevo valor que le vamos a dar a la propiedad.

Consideraciones

Al conjunto propiedad:valor se le llama declaración
Al conjunto selector+declaración se le denomina regla
