Formas de incluir CSS en un documento HTML

Estas son las diferentes formas de incluir CSS en HTML:

  • Usando el método link en la etiqueta <head>

    • Este es el método más recomendable, consiste en incluir la etiqueta <link> con los parámetros "rel" y "href" donde el primero se referirá a que se trata de una hoja de estilos y el segundo la dirección del archivo, con extensión .css, que contiene los estilos.
    La forma de escribirse es la siguiente:

    <link rel="stylesheet" href="\css\style.css">

  • Usando la etiqueta style dentro de la etiqueta <head>

    • Para este caso el estilo es aplicado dentro de un documento HTML y solo dentro del mismo, un ejemplo sería el siguiente:

            
          <style>
            body {
              background-color: slateblue;
            }
          </style>

    Solo se debe usar en casos muy concretos en donde no se puede tener un archivo externo, por ejemplo en un correo electrónico

  • Utilizar CSS en línea

    • Se utiliza el atributo style dentro de una etiqueta, se considera muy mala práctica, se suele usar en casos de estilos dinámicos con JavaScript.

    
        <h2 style="background-color: teal" >
  • Usar la regla import dentro de la etiqueta <head>

    • Funciona perfectamente en general, sin embargo es más tardío y la carga es asincrónica, es decir, la página cargará y mientras tanto se aplicarán los estilos. La página se verá de una forma y una vez los estilos estén listos, se verán. No recomendado.

    
        @import url(css/styles.css);
